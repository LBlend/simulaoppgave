# Oppgave for Disputas AS

Laget med `create-react-app`

## Installasjon

`yarn install`
eller
`npm install`

## Kjør

`yarn run start`
eller
`npm run start`

## Bygg

`yarn run build`
eller
`npm run build`

## Deploy til Github Pages

Her må du sørge for at den riktige URL-en, markert som `homepage`, ligger i [package.json](package.json) filen-

`yarn run deploy`
eller
`npm run deploy`

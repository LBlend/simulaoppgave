# Simulaoppgave - Oppgave for Disputas AS

Laget med `create-react-app`

## Installasjon

```bash
yarn install
```

eller

```bash
npm install
```

## Kjør

```bash
yarn run start
```

eller

```bash
npm run start
```

## Bygg

```bash
yarn run build
```

eller

```bash
npm run build
```

## Deploy til Github Pages

Her må du sørge for at den riktige URL-en, markert som `homepage`, ligger i [package.json](package.json) filen

```bash
yarn run deploy
```

eller

```bash
npm run deploy
```
